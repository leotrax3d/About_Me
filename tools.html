Tools

small, single-purpose browser tools — no build step, no server, nothing leaves your machine. each one is a self-contained page.

tools

QR Generator
turn any text or URL into a QR code. pick the size, error correction and colours, then download as PNG, JPG or SVG.
qr · png · svg
URL Hider
generate a forwarding link that hides its destination. choose a redirect delay or require a confirmation click. includes a QR code and an inspect mode.
redirect · base64
Checksum
compute SHA-1 / 256 / 384 / 512 hashes of text or a dropped file via the Web Crypto API, and verify a download against an expected checksum.
sha · verify
Base64
encode text to Base64 and decode it back, with full UTF-8 support and a data-URI mode for small files. runs entirely in your browser.
encode · decode
