classchat
offline messaging

lightweight local network chat application built for schools, classrooms and small shared environments. no server. no accounts. no installation.

01

overview

classchat is a simple local chat system that works through a shared network folder.

instead of using a central server, all messages, channels, attachments and status information are stored inside a common directory that every participant can access.

the project was designed for classrooms, computer labs and educational environments where a fast communication tool is needed without complicated setup.

users can join channels, exchange files, view online members and communicate in real time across the local network.

03

getting started

step 1
download the latest release package and extract the archive.
download
step 2
run start.bat. the batch file automatically launches the powershell based chat interface.
launch
step 3
select a shared network folder that all users can access. classchat creates the required folders and files automatically.
setup
step 4
choose a username, join a channel and start chatting with everyone connected to the same shared folder.
ready
04

how it works

classchat stores messages, channels, user status information and file attachments inside a shared folder. every client reads and writes to this location, allowing communication without requiring a dedicated server.
